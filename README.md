<h2>Proyecto Base de Datos FASE 2</h2>
<b>Este proyecto es una secuencia de scripts de sql que se basan en varias etapas de la creacion de las bases de datos</b>
<br>

<p>Donde tendremos los siguientes .sql </p>
<ol>
  <li>DDL.sql (Creación de tablas).</li>
  <li>Inserts.sql (Carga de datos).</li>
  <li>Programmability.sql (SPs, Funciones, Triggers).</li>
  <li>Queries.sql(Consulotas de reporte).</li>
</ol>
